# OSChina Android [客户端](http://www.oschina.net/app/)

##写在前面的话
基于官发开源的代码改写并优化，使用RxJava/RxAndroid/RxBus/RxPermission/Dagger2/Retrofit/OkHttp/Picasso/ButterKnife/XStream(SimpleXML)作为基础库简化代码的编写。
